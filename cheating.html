Cheating

Keterangan

Ridhwan menyukai cheat karena dapat membuat game-game menjadi lebih mudah dan lebih seru.

1. Menggunakan Cheat Engine.

Cheat Engine alat untuk mengubah sistem dalam program,dapat lebih mudah jika menggunakan tables.

Gambar-1 : Tampilan pada aplikasi Cheat Engine.

2. Mengubah data dalam file.

Mengubah suatu data dalam file memiliki resiko errornya aplikasi tapi jika sudah menguasai maka game tersebut sudah berada dalam kendalimu.

Gambar-2 : File yang akan diubah datanya.

Pertama-tama adalah pilih File yang akan diubah datanya,yang akan diubah kali ini adalah jumlah uang yang ada pada game bernama the Guild 2.

Gambar-3 : Tampilan game pada sebelum diubah datanya.

Jumlah uang terdapat pada bagian tengah atas,sekarang jumlahnya masih 0.

Gambar-4 : File yang sudah diubah datanya.

Sekarang kita ubah angka pada bagian tulisan Money untuk menambahkan uang.

Gambar-5 : Tampilan game yang sudah diubah datanya.

Sekarang jumlah uangnya sudah berubah.
